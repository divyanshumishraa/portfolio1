Manufacturing UI

Project Overview

This project showcases a responsive UI for a manufacturing dashboard, built using HTML, CSS, and JavaScript, with a Spring Boot backend for API integration.

Features

  • Responsive layout with grid-based design
  • Interactive charts and tables
  • REST API integration with Spring Boot
  • Dark mode toggle
